Mind map studio

One shared canvas for Full map, One topic, and Customize. Load the live published JSON, or import a draft file. Export anytime for backups. Staff publish site-wide changes from Site admin → Content.

How to use this page
  • Reload live graph — downloads the canonical tree from the site (same as the home page). Requires network; uses your session only for merged user progress.
  • Export / Import — file is the mind map array (root topics), not your personal overlay file. Import replaces the tree in the browser; it does not publish to everyone unless you use admin CMS.
  • View tabs — switch between the full hierarchy, a single topic, and customize mode (if your account has access).
  • Toolbar above the map — expand all, collapse all, zoom; plus map export/import on the left of that bar.
  • First visit — a short guided tour opens automatically once; use Start tour anytime.

Prefer a lighter entry? Use the home page for the same graph with a shorter intro.
