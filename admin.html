Site settings

Remote key/value config in app_kv. Values are strings (often JSON). Save writes one key; Delete key removes it from D1. JSON tab can bulk-apply many rows.

User-facing features

Controls what all visitors and members see on the public site (practice map, tabs, dashboard link, sign-in, OAuth buttons, footer, member hub modules, etc.). Stored as JSON under key site_user_features. Enabled = the feature works when reached; Visibility = it appears in navigation and other UI. Turning Enabled off automatically turns Visibility off for that row. Use Save to server to persist, Reload from server to refresh the matrix (drops unsaved edits).
